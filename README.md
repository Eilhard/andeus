# Andeus
RESTful API & Front-end client for tabletop role-playing game website.

## Technologies used
  * Front-end
    * SCSS
    * VueJS (Vuex)
    * Webpack
  * Back-end
    * NodeJS
    * ExpressJS
    * MongoDB

## Notes
 Backend files config.js and guard.js added to gitignore for security reasons.
